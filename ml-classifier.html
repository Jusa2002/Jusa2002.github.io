Machine Learning Classifier

My first Java project — a classifier built from scratch without ML libraries.

Overview

A classifier is a function that maps an input — a vector of measurements — to a predicted class. The challenge is learning that function from data. A naive approach would be to estimate the full joint probability distribution of all variables, but this runs into a fundamental problem: the domain grows exponentially with the number of features, requiring an impractically large dataset to avoid overfitting.

This project addresses that by using a Markov Random Field (MRF) — a probabilistic graphical model that encodes conditional independence assumptions between variables, drastically reducing the number of parameters needed. The classifier was built around the Chow-Liu algorithm, which learns the optimal tree-structured MRF from data by finding the maximum spanning tree over pairwise mutual information scores. Classification then reduces to finding the class that maximises the joint probability under the learned model — a maximum likelihood estimate.

The implementation was done entirely in Java from scratch, without any machine learning libraries. It was trained and tested on biomedical datasets from the UCI Machine Learning Repository, including a cancer dataset where the goal was to classify tumours as benign or malignant based on ten clinical measurements.

This was my first substantial Java project, and it gave me a solid grounding in both probabilistic reasoning and practical software design — working with matrices, graph algorithms, and data pipelines all within the same codebase.
